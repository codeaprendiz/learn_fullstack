## Keyboard shortcuts

### VS Code (mac)

- Toggle between full screen and last open window size
  - `cntr + cmd + F`

- To open terminal
  - `cntr + ~`

- To switch terminals in VS Code
  - `CMD + Shift + [` or `CMD + Shift + ]` 

- To open first file in editor
  - `cntr + 1`
