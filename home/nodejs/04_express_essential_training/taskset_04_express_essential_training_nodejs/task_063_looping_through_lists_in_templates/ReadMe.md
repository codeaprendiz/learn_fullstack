# Looping through lists in templates

```html
                        <% speakerNames.forEach( (speaker)=> { %>
                            <a class="dropdown-item" href="/speakers/<%= speaker.shortname %>">
                                <%= speaker.name %>
                            </a>
                            <% }) %>
```
